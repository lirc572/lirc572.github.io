## Intorduction

Setting up a development environment for ESP32 (using Espressif SDK) with CLion.

- Download and build esp-idf
- Install CLion
- Install *ESP32* plugin
- Configure ESP32 plugin
- Start a simple project (hello world) and upload to ESP32
